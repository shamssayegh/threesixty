# Three Sixty Landing Page

One-page static landing page based on the provided design reference.

## Files
- `index.html`
- `styles.css`
- `main.js`
- `assets/logo.svg`

## Local Preview
Open `index.html` in your browser.

## Hostinger Deployment (public_html)
1. Log in to Hostinger and open your hosting dashboard.
2. Go to **Files** > **File Manager**.
3. Open the `public_html` directory.
4. Upload `index.html`, `styles.css`, `main.js`, and the `assets/` folder (including `logo.svg`).
5. If you already have files in `public_html`, replace them or move them to a backup folder first.
6. Visit your domain to confirm the page loads.
